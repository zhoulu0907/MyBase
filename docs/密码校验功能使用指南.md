## 密码校验功能使用手册

### 应用场景

- 注册/找回密码/修改密码/重置密码等：对用户输入做强度校验并给出明确提示。

---

### 一、使用方式

#### 2.1 方式一：同模块编程式调用

需要在业务逻辑中主动调用校验。

**Service示例**：
```java
@Service
public class UserRegisterServiceImpl {
    
    @Resource
    private PasswordValidationService passwordValidationService;
    
    public void register(UserRegisterReqDTO reqDTO) {
        // 主动调用密码校验
        try {
            passwordValidationService.validatePassword(reqDTO.getPassword());
        } catch (WeakPasswordException e) {
            throw exception(USER_PASSWORD_WEAK, e.getMessage());
        }
        
        // 继续处理注册流程...
    }
}
```

#### 2.2 方式二：跨模块Feign调用

其他模块需要校验密码时，通过Feign调用。

**调用示例**：
```java
@Service
public class ExternalServiceImpl {
    
    @Resource
    private PasswordValidationApi passwordValidationApi;
    
    public void createUser(UserCreateReqDTO reqDTO) {
        // 通过Feign调用密码校验API
        CommonResult<Void> result = passwordValidationApi.validatePassword(reqDTO.getPassword());
        if (!result.isSuccess()) {
            throw new ServiceException(result.getMsg());
        }
        
        // 继续处理...
    }
}
```
