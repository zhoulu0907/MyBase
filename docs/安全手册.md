1. 租户级安全配置项
    - 配置位置：infra_security_config，过滤 tenant_id = 目标租户ID的数据，缺失的配置项则使用 infra_security_config_template 中默认配置。
    - 全量配置位置：redis缓存，infra:security:tenant-config:租户ID
2. 密码错误次数过多被锁定
    删除redis缓存： infra:security:login:lock:租住ID:用户ID
3. 更新密码强度：
   - a. infra_security_config，过滤 tenant_id = 目标租户ID； 
     - 不校验弱密码： enableWeakPassword = false；
     - 密码最小长度：minLength= N；
     - 密码错误重试次数：failedLockThreshold = N；
   - c. 清除redis缓存后生效：infra:security:tenant-config:租户ID
