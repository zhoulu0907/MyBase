## 密码校验功能使用手册

### 应用场景

- 注册/找回密码/修改密码/重置密码等：对用户输入做强度校验并给出明确提示。

```java
        CommonResult<Boolean> result = passwordValidationApi.validatePassword(tenantId,reqDTO.getPassword());
        
        // 继续处理...
    }
}
```
